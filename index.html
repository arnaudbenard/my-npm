NPM Stats

The cool utility app to monitor the popularity of your npm modules.

Why this project?

After multiple attempts to learn iOS development, I decided to follow Paul Hegarty's iPhone Application Development class and build a native app. In a few weeks, I have been able to learn how to work with Swift and build this app. In the spirit of openness, I have decided to make all the code open-source on GitHub. Feel free to send me feedback and report issues.
